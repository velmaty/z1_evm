# Генератор заданий по машинному представлению вещественных чисел

## Описание
Программа генерирует индивидуальные задания для студентов.

## Компиляция
```bash
gcc -o z1.c -lm
```

## Использование
```bash
./z1               # ручной ввод
./z1 config.txt   # из файла
```

## Параметры
- n - количество вариантов
- k - заданий на студента  
- разрядность (32/64)
- a b - диапазон чисел
- p - знаков после запятой

## Пример config.txt
```
5 10 64 -100 100 3
```
